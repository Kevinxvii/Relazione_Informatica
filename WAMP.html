WAMP

Il termine WAMP si riferisce a un framework di sviluppo per applicazioni web e web service. È stato introdotto come alternativa al più popolare stack LAMP, che include Apache, MySQL e PHP. Componenti installati:

  • Apache: il Web server;
  • MySQL: il database management system;
  • PHP, Perl e/o Python: i linguaggi di scripting.
